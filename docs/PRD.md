# 아띠끄디자인(Attique Design) 웹사이트 PRD v3.0

> **v3.0 핵심 변경:** 기존 커머스 + 우측 세로 네비게이션 구조 → **하이엔드 에디토리얼 매거진** 스타일로 전면 전환.
> - 햄버거 메뉴 기반 풀스크린 오버레이 네비게이션
> - Landing 섹션에 배경 비디오 + 스크롤 드리븐 로고 애니메이션
> - 에디토리얼 Overlapping 레이아웃
> - 공간 전시(Exhibition Archive) 기반 Selection
> - 극단적 미니멀 Contact 섹션

## 1. 개요 및 목표

### 1.1. 제품 정의
"하이엔드 홈스타일링부터 프리미엄 인테리어 시공까지, 통합 토탈 리빙 솔루션"
한남동, 압구정 일대 고급 주거지 거주자 및 프리미엄 1~2인 가구를 위한 토탈 리빙 및 공간 디자인 편집샵 플랫폼. 하이엔드 가구 전시 및 판매, 그리고 1:1 프라이빗 공간 설계/시공 컨설팅을 통합 제공합니다.

### 1.2. 디자인 비전 (v3.0 신규)
아띠끄 디자인 웹사이트는 **하이엔드 에디토리얼 매거진**의 감각을 디지털로 번역한 플랫폼입니다. 정적인 커머스 페이지가 아닌, 인쇄 매거진을 넘기는 듯한 동적이고 감각적인 경험을 제공합니다.

**핵심 레퍼런스:**
- **miltontextiles.com** — 스크롤 드리븐 로고 애니메이션
- **rayphilly.place** — AMENITIES 섹션 비대칭 그리드

### 1.3. 솔루션 및 가치 제안
- **에디토리얼 브랜드 경험:** 배경 비디오, 오버래핑 타이포그래피, 비대칭 그리드를 통한 감각적 몰입 경험.
- **쇼룸 방문 유도:** 제품 가격 노출 대신 'Enquire' / 'View Detail' 버튼으로 프라이빗 쇼룸 방문을 자연스레 유도.
- **감도 높은 네비게이션:** 얇고 세련된 햄버거 메뉴 → 딥 버건디 풀스크린 오버레이로 브랜드 정체성 강화.

## 2. 글로벌 네비게이션 및 스크롤 (Navigation & Scroll)

### 2.1. 스크롤 방식
- **원페이지 스크롤 스냅핑:** `scroll-snap-type: y mandatory` + 100vh 섹션 유지.
- **섹션 간 이동:** `scroll-behavior: smooth` + `ease-in-out` 가속도 커브 적용.
- **브라우저 기본 스크롤바 숨김.**

### 2.2. 햄버거 메뉴 (Hamburger Menu)
- **위치:** 우측 상단 고정 (`position: fixed`, `top: 2rem`, `right: 2rem`)
- **디자인:** 극도로 얇은 선 2~3줄. 흰색/검정 자동 전환 (배경 밝기에 따라).
- **클릭 시 동작:**
  1. 딥 버건디(`#8B0000`) 배경의 풀스크린 오버레이 메뉴가 열림.
  2. 각 섹션(0~4) 이름이 거대한 타이포그래피 링크로 배치됨.
  3. 링크 호버 시 텍스트 컬러가 크림/골드로 전환 + 미세한 위치 이동.
  4. 클릭 시 해당 섹션으로 부드러운 스크롤 이동 + 오버레이 닫힘.
- **메뉴 항목:**

| 순서 | 메뉴 라벨 | 대응 섹션 | 섹션 ID |
|------|-----------|-----------|---------|
| 0 | Landing | Section 0: 배경 비디오 + 로고 | `#intro` |
| 1 | Philosophy | Section 1: 브랜드 철학 | `#philosophy` |
| 2 | Portfolio | Section 2: 포트폴리오 큐레이션 | `#portfolio` |
| 3 | Selection | Section 3: 공간 전시 아카이브 | `#selection` |
| 4 | Contact | Section 4: 미니멀 컨택트 | `#contact` |

### 2.3. 기존 VerticalSideNav → 폐기
v2.0의 우측 세로 고정 네비게이션은 **삭제**하고, 햄버거 메뉴 오버레이로 대체.

## 3. 메인 랜딩 페이지 (One-page Scroll Snapping — 5 Sections)

### 3.0. Section 0: Landing (Video & Logo Animation)

#### 배경 비디오
- 첫 화면에 배경 비디오를 `object-fit: cover`로 꽉 차게 배치.
- PC/모바일 화면비에 맞춰 잘리지 않도록 최적화.
- `<video autoplay muted loop playsinline>` 속성 적용.
- 임시 단계에서는 고급스러운 인테리어 무드의 Placeholder 비디오(Pexels/Coverr 등) 사용.

#### 로고 애니메이션 (Scroll-driven)
- **랜딩 시:** 화면 정중앙에 거대한 로고 (`ATTIQUE DESIGN`) 배치. 로고는 딥 버건디(`#8B0000`) 배경에 넓은 자간의 흰색 텍스트(Raleway 폰트)로 구성.
- **스크롤 시작:** 로고가 점진적으로 크기가 작아지며 상단 좌측(또는 중앙)으로 이동 → Sticky 고정.
- **구현:** CSS `position: fixed` + JavaScript scroll 이벤트 기반 `transform: scale()` + `translate()` 인터폴레이션.
- **참고:** miltontextiles.com의 로고 축소 + 고정 효과.

#### 스크롤 힌트
- 하단에 미세한 "Scroll" 텍스트 + 선 애니메이션.

### 3.1. Section 1: Philosophy (Overlapping Editorial Design)

#### 타이틀
- **"When Preference Becomes *Lifestyle*."**
- `Lifestyle`은 딥 버건디(`#8B0000`) + 이탤릭체 (Playfair Display Italic).
- 섹션 상단에 거대하게(60~80px) 배치, 아래 본문과 레이어가 살짝 겹치는 에디토리얼 스타일.

#### 본문 텍스트
- 타이틀 아래로 충분한 여백 후 Pretendard로 브랜드 철학 서술.
- 타이틀과 본문 사이 오버래핑(Overlap) 레이아웃 적용.

#### 서비스 영역 나열
- Interior, Styling, Furniture, Lighting, Fabric, Rug, Accessories 등을 나열.
- **표현 방식:** 뻔한 목록이 아닌, 화면 하단에 아주 작은 폰트 + 얇은 세리프체 + 넓은 자간으로 가로 흐름 텍스트 또는 세로 텍스트 애니메이션 적용.

### 3.2. Section 2: Portfolio (Curated Selection)

#### 비대칭 그리드
- rayphilly.place의 AMENITIES 섹션 참고.
- 4대 카테고리: **Drama, Magazine, Residential, Commercial** 대표 컷 배치.
- 사진 크기 비대칭: Residential·Commercial은 크게(70%+), Drama·Magazine은 작게 배치.

#### 가독성 최적화
- 사진 위 텍스트: 전체 디자인 테마에 맞추어 `Playfair Display/Raleway` 및 밝은 트랜스루슨트(반투명 차콜브라운) 배경 등으로 폰트 컬러와 스타일, 배치 가독성을 높임.

#### 페이지 전환 및 상세 페이지 (포트폴리오)
- 클릭 시 상세 포트폴리오 페이지로 전환.
- **포트폴리오 상세 레이아웃:** 기존 좌측의 세부 정보 4개 섹션(01-04 등)을 제거. 페이지 상단에 공간명(검정색 소제목 크기 폰트)과 하단에 설명글 배치. 그 아래 가로 스크롤(Horizontal Scroll) 가능한 사진 갤러리 형태로 구성.
- 최하단 연락처 정보(Contact Footer) 영역은 Section 4와 동일한 정보(인스타그램, 블로그, 카톡)를 바탕으로 심플한 디자인으로 삽입.

### 3.3. Section 3: Selection (Exhibition Archive)

#### 컨셉: "제품 나열이 아닌 공간 전시"
- **좌측:** 스타일링된 공간 사진을 최대 3개까지 배치. 제어는 우측 하단의 페이지네이션 닷(dot)으로만 수행 (좌측 중앙 스팟 제거). 각 공간별 3개의 사진으로 통일하여 3개의 닷 노출.
- **우측:** 해당 공간에 쓰인 오브제들과 제품 사진들을 잘림 없이 상하 여유롭게 배치, 위아래로 자유롭게 스크롤하며 볼 수 있도록 구현.
- **동적 관리:** 스타일링 샷과 제품 리스트는 향후 사이트 관리자가 지속적으로 추가 및 삭제할 수 있도록 동적으로 데이터 바인딩되도록 설계(JSON 등 구조화).
- **CTA 버튼:** 제품 버튼은 `View Detail` 단일로 통일. 클릭 시 해당 제품 상세 페이지로 이동하며, 이 페이지는 밝은(흰색계열) 배경 위에서 제품 사진이 크고 잘 보이도록 구성(좌 사진, 우 텍스트 또는 텍스트 병기 형식).

### 3.4. Section 4: Contact & Booking (Minimalist Finish)

#### 디자인
- **에디토리얼 스플릿 레이아웃:** 
  - 좌측: 첨부된 아띠끄 쇼룸 빌딩 전면 사진 노출 (`aspect-ratio: 3/4` 잡지 에디토리얼 비율).
  - 우측: 광활한 여백 중앙에 타이포그래피 배치.
- 메인 타이틀: **"Private Consultation."** — 딥 버건디(`#8B0000`), Playfair Display Italic.
- 서브 텍스트: 쇼룸 주소 (서울시 용산구 한남대로 21길 27 아띠끄 디자인) 및 전화번호(02-3443-8170) 노출. (폰트 컬러: `#000` 완전한 블랙, 굵기 400)

#### 카카오톡 문의
- 전화번호 바로 옆에 **카카오톡 아이콘** 노출, 클릭 시 카카오톡 채널 상담으로 연결. 

#### 네이버 예약
- **"방문상담 예약하기"** 버튼 노출, 클릭 시 네이버 예약 외부 URL 연결.

#### Footer 통합 (국내 규정)
- 사이트 최하단에 사업자 정보(상호, 대표자명, 사업자등록번호, 통신판매업신고번호, 주소, 개인정보관리책임자 등) 및 저작권 표기를 가장 작게 배치.

## 4. 상세 기능 명세 (별도 라우트 기능)

> 아래 기능들은 원페이지 랜딩 바깥의 별도 라우트(`/products`, `/products/:id`, `/login` 등)로 운영됩니다.

### 4.1. 회원가입 및 마이페이지 (Auth & My Page)
비회원도 제약 없이 탐색과 액션이 가능하도록 설계하되, 자연스러운 가입을 유도.
- **회원가입/로그인:** 이메일 기반 가입 및 소셜 간편 로그인 (카카오, 네이버, 애플 등) 지원.
- **마이페이지:** 주문/배송 조회, 위시리스트, 1:1 문의 내역, 방문 상담 예약 현황.

### 4.2. 상품 탐색 및 상세 (PLP & PDP)
- **상품 목록 페이지 (PLP):** 정렬/필터링, 갤러리 스타일 레이아웃.
- **상품 상세 페이지 (PDP):** 모듈형 에디터, 희소성 뱃지, 구매 액션 분리.

### 4.3. 통합 1:1 프라이빗 방문 상담 예약 (Dynamic Booking Form)
- Step 1. 상담 분야 선택 (Radio): 인테리어 설계 및 시공 / 가구 및 홈스타일링 / 토탈 솔루션
- Step 2. 조건부 입력 폼 활성화

### 4.4. 관리자 시스템 (Admin / CMS / CRM)
- 권한 및 메뉴 제어, 상품 에디터, 재고 관리, 통합 CRM 파이프라인.

## 5. 기술 및 디자인 요구사항 (Non-Functional)

### 5.1. UI/UX 디자인 가이드
- **하이엔드 에디토리얼 매거진 스타일:** 인쇄 매거진을 넘기는 듯한 레이아웃. 텍스트 오버래핑, 비대칭 그리드, 과감한 여백.
- **원페이지 스크롤 스냅핑 UX:** ease-in-out 가속도, 모바일 터치 스와이프 동일 경험.
- **햄버거 메뉴 오버레이:** 딥 버건디 풀스크린 오버레이, 거대 타이포그래피 링크.

### 5.2. 반응형 웹 디자인 최적화 (Responsive Web)
- **햄버거 메뉴:** PC/모바일 동일 경험.
- **배경 비디오:** `object-fit: cover` 기반, 화면비에 따른 자동 크롭.
- **포트폴리오 그리드:** PC(비대칭 4칸) → 모바일(세로 2열 또는 1열).
- **Selection 레이아웃:** PC(좌 공간샷 + 우 오브제) → 모바일(세로 스택).
- **모바일 폼 UI:** 터치 영역 최소 44px 이상 확보.

### 5.3. 보안 및 인프라
- SSL 인증서 발급 및 DB 암호화 필수 적용.
